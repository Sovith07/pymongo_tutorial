# 📝 MongoDB Todo CLI Application

A simple command-line Todo Manager built with **Python** and **MongoDB** using **PyMongo**. This project demonstrates basic CRUD database operations and MongoDB integration with Python.

---

## 🚀 Features

- ✅ Create new tasks
- 📋 View all tasks
- 💾 Store tasks permanently in MongoDB
- 🔄 Real-time database interaction using PyMongo
- 🎯 Beginner-friendly project for learning MongoDB

---

## 🛠️ Tech Stack

- Python 3.x
- MongoDB
- PyMongo
